# WattMind - Predictive Energy Sizing Tool 

AI-ready energy analysis and system sizing engine for modern distributed power systems.

> GridSense estimates electricity consumption, forecasts solar generation, and recommends optimal energy system configurations for residential, commercial, and off-grid applications.
> The edge is predefined archetypes of consumer types and modular/flexible access to load selection and typing. Calculator seamlessly integrates 1P 230V AC, 3P 400V AC and 12/24/48V DC load types.
> Intended for non-technical everyday users and config access for expert users as well.

---

## 🎯 Problem

Energy system design is often:
- manual and time-consuming  
- dependent on rough assumptions  
- disconnected from real consumption behavior  

This leads to:
- oversizing (high CAPEX)  
- undersizing (system failure risk)  
- inefficient system configurations  

---

## 💡 Solution

GridSense Engine provides a structured, data-driven approach to:

- Estimate load profiles  
- Simulate solar generation potential  
- Optimize system sizing (battery, inverter, PV)  

It acts as a **predictive layer between raw energy data and system design decisions**.

---

## ⚙️ Key Features

- 📊 Load estimation based on appliance modeling  
- ☀️ Solar generation forecasting (location-based potential ready)  
- 🔋 Battery sizing logic (capacity vs autonomy tradeoff)  
- 🔌 Inverter sizing recommendations  
- 🧠 AI-ready architecture for future predictive enhancements  
- 📁 Dataset-driven modeling approach
